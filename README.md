# PhotoMemo

SwiftUI iOS app scaffolding using the Swift Package Manager.

## Features
- Photo picking via `PhotosPicker` limited to still images.
- Multiline note entry with adjustable font size and editable timestamp.
- Live preview of the composited photo, timestamp, and note with drop shadow.
- Save the rendered image to the photo library or share using the system share sheet.
- Permissions use read–only access for picking and add–only for saving.

Build the app with Xcode 15 or later using the package manifest.
